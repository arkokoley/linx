# Installation Instructions

Dependencies to install:

1. node.js above v6.3.0  
2. npm above 3.10.3  
3. `npm install -g yarn`  
4. MySQL  


Commands to run:  

1. `git clone git@github.com:arkokoley/linx.git`  
2. `yarn install`  
3. Rename `sample.env` to `.env` and fill the variables  
4. `npm run dev`
